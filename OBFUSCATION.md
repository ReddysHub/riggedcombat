# Obfuscation Disclosure

The following tools and methods were used to obfuscate the distributed JAR file:

1. **ProGuard**: Used for basic class/member renaming and package flattening.
2. **Skidfuscator**: Used for control-flow obfuscation and string encryption.
3. **Custom Hardener**: Used for JAR entry shuffling and decoy class injection.

Clean source code is provided in this repository for review.
